# importer

Wrap → share, conversationally (ARCHITECTURE §6): inventories your harness (guided by
the cheat-sheet's Introspection section), clusters artifacts into the use case you
named, maps them to package primitives, splits generic mechanism from personal nuance,
and emits `capabilities/<id>-draft/` + a draft `MOD.md` + `GAP.md`. Read-only on the
live setup; secrets flagged, never copied; the PR stays yours to open.

Spec one-pager: [importer.md](https://github.com/AlmogBaku/aos/blob/spec/capabilities/importer.md)
